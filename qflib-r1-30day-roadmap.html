QF-Lib × Trading-R1 · 30일 실행 로드맵

한 달 안에 “LLM 시그널을 qf-lib에서 검증”하는 파일럿 만들기

목표는 Trading-R1 전체 재현이 아니다. 먼저 qf-lib 위에서 LLM이 종목/weight/근거를 뱉고, 그 결과를 look-ahead 없이 백테스트·라이브 섀도우로 검증하는 최소 시스템을 만든다.

Track A: qf-lib 실증 MVP
Track B: LLM 학습 / LoRA SFT
Karpathy nanoGPT/nanoChat 포함

30일 후 우리가 가져야 할 것

qf-lib 위에서 Prompt-only LLM과 작게 학습한 SFT 모델이 모두 weight+rationale을 출력하고, 둘을 baseline과 함께 backtest + live shadow로 비교할 수 있어야 한다.

Track A

qf-lib 실증 MVP

  • MarketSnapshotBuilder
  • LLMWeightProvider
  • qf-lib backtest integration
  • rationale 포함 HTML report
  • live shadow paper portfolio
Track B

LLM 학습 역량 + R1-like SFT

  • Karpathy nanoGPT/nanoChat 튜토리얼
  • Qwen/Llama small model inference
  • LoRA SFT 환경 구축
  • signal+rationale JSON 학습
  • schema-following eval

가장 중요한 원칙

  • Pretraining은 하지 않는다. 한 달 목표는 pretraining이 아니라 LoRA SFT다.
  • RL/RFT는 Month 2로 넘긴다. 첫 달에 RL까지 욕심내면 데이터/검증이 무너진다.
  • LLM은 봉인된 snapshot만 본다. qf-lib가 준 point-in-time 입력 외부를 조회하지 못하게 한다.
  • 성과보다 검증 인프라가 먼저다. “돈 번다”가 아니라 “엄밀히 검증된다”가 첫 달의 산출물이다.
Week 1

qf-lib MVP + LLM 학습 튜토리얼

qf-lib

SPY, QQQ, AAPL, MSFT / Yahoo OHLCV / 기본 기술지표 / MarketSnapshotBuilder / LLMWeightProvider.

LLM

Karpathy nanoGPT 또는 nanoChat 1회 완주. tokenization, pretraining, SFT, sampling, eval 감각 확보.

완료

LLM weight로 qf-lib backtest 1개 실행. rationale과 성과가 같은 리포트에 찍힘.

Week 2

Point-in-Time Dataset Builder

Data

OHLCV, 기술지표, FRED macro, SEC companyfacts/EDGAR. 뉴스는 optional로 Google News headline부터.

Build

ticker-day snapshot, as_of_date 필터, snapshot hash, parquet/jsonl 저장, 2021~현재 기간.

완료

임의 날짜 t의 입력에 t 이후 정보가 없음. 같은 snapshot은 항상 같은 hash.

Week 3

Label/Reward + SFT v0

Label

3/7/15일 forward return, 20일 vol normalize, 0.3/0.5/0.2 weighted score, 5-class action.

SFT

Qwen 1.5B~4B 또는 Llama 계열 LoRA SFT. rationale은 v0에서 template로 시작.

완료

JSON schema 95%+ 준수. Buy/Sell/Hold 분포 붕괴 없음. qf-lib에 바로 투입 가능.

Week 4

Backtest 비교 + Live Shadow

비교

equal weight, momentum/risk parity, prompt-only LLM, SFT v0를 qf-lib에서 동일 조건 비교.

Live

매일/매주 snapshot 생성, weight 출력, 실제 주문 없이 paper portfolio 운용.

완료

backtest report HTML, live shadow dashboard 초안, CERN/qf-lib용 1-page memo.

최종 산출물 체크리스트

1. qf-lib integration
MarketSnapshotBuilder · LLMWeightProvider · rationale 포함 backtest report
2. Dataset
point-in-time ticker-day snapshots · label/reward generator · train/valid/test split
3. LLM 학습
nanoGPT/nanoChat 실습 완료 · LoRA SFT v0 checkpoint · schema-following eval
4. 검증
prompt-only vs SFT vs baseline qf-lib backtest · live shadow mode 시작
5. 제안 자료
“R1-style reasoning signal을 qf-lib에서 재현 가능하게 검증하는 오픈소스 파일럿”으로 정리
