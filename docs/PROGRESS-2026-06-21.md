**Can the IC cap push further? YES — via prediction HORIZON, not features.** Rich features
(80, 3× the baseline) + cross-sectional rank-of-everything + market-neutral target all stay
**stuck at OOS IC ~0.23** (the ~0.24 price/technical cap is firm against feature/framing

horizons (2,3,5)→**IC 0.333**, (3,7,15)[current]→0.235, (5,10,21)→0.227, (10,21,63)→0.188.
Short-horizon returns are far more predictable (momentum/mean-reversion); the current
3/7/15-day label (paper-matched) is a *harder target than necessary*. **BUT — the IC gain
does NOT survive transaction costs** (10 bps/side, top-6 long, 2024-26): (2,3,5)d at rebal/3d
→ NET SR +0.36, at rebal/5d → +0.75, while the current **(3,7,15)d weekly → NET SR +1.14**
(CR +46 %, MDD 12 %, the best of anything incl. graded LLM 0.93). Short horizons decay fast →
need faster rebal → turnover eats the higher IC. **Net: the *IC* cap pushes further via
horizon, but the *usable-alpha* cap is cost-bound — the current 3/7/15 weekly is the
net-Sharpe sweet spot.** (Side note: a pure GBM beats every LLM here — GBM>LLM reconfirmed.)
