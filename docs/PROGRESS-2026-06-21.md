## Update 2026-07-03 — ceiling investigation OVERTURNS the "0.21 input ceiling" claim

Full write-up + literature: [`2026-07-03-why-the-ceiling.md`](2026-07-03-why-the-ceiling.md)
(experiments `compare_lab/ceiling_probe.py`). Self-correcting result:

- **"0.21 = input ceiling, LLM captures 76 %" was WRONG.** A one-feature **linear momentum**
  model scores **IC 0.266** on `make_signal` — above GBM (0.215) and the base LLM (0.205), which
  *under-extract*. The label is momentum-dominated; extra features add noise.
- **The proxy is ~4× inflated by smoothing.** Re-scored against the **raw 7-day return**,
  momentum IC collapses **0.266 → 0.064** (GBM 0.042). Everything reported as IC 0.16–0.27 is the
  predictability of a smoothed EMA/overlapping-horizon artifact (Lo–MacKinlay 1988), not tradeable
  skill. **Real tradeable IC ≈ 0.06** — the weak-form-EMH floor.
- **Two ceilings:** proxy ≈0.27 (a *model/extraction* gap, in principle closeable) vs real-return
  ≈0.06 (an *information* limit — DPI + EMH — closeable by no model). In-sample R² is only 0.16
  (features explain ~16 % of the label) → the rest is irreducible noise.
- Literature grounds it: DPI/Bayes-error (Cover–Thomas, Fano), EMH (Fama; Grinold), smoothing
  inflation (Lo–MacKinlay), RL-bounded-by-base (2504.13837, 2510.15990), LLM-no-TS-benefit
  (2406.16964, 2404.11757). Honest scope: "0.06 = I(X;Y)" is our *inference*; the direct MI
  estimate (E7) and a tabular alt-data ablation for #2 (E8) are named as the proofs-in-waiting.
