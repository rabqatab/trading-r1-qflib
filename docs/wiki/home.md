## Conventions

Pages are declarative (what is known), dated docs are evidentiary (how we know). If a wiki page
and a dated doc disagree, the wiki page is newer — but file an inconsistency, don't silently
trust either. Status values: `established` (multiple independent confirmations),
`closed-negative` (tested, null, don't retry without new data), `open` (gated, not abandoned).
